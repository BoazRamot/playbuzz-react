## PlayBuzz-App 
## Description
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Project uses, React, Redux, React-Router, MATERIAL-UI.

This project using localStorage Api - On each state change, save it to local storage.

This project contains 3 quizzes (3 json files) and Fetch them.

A Progress bar show the user's progression. 

## Flow

1. User select a quiz

2. First question presented for the selected quiz

3. User select click an answer to go next question

4. After user answered all questions

- a summary of results is presented

- User can choose to redo quiz / select different quiz

** If user reload page - it will recover last saved state

** Random questions order

## Usage
1.Run `git clone https://github.com/BoazRamot/playbuzz-react.git`

2.Open repo  `cd playbuzz-react`

3.Run `npm I`, to install the module's prerequisites

4.Run `npm start`, to run the App.

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
